# Mini NPU Simulator

NumPy, Pandas 등 외부 라이브러리를 일절 배제한 Pure Python 기반 MAC 연산 시뮬레이터 및 성능 분석 프로젝트.

## 1. 아키텍처 설계 배경: "Inside-Out"
본 프로젝트는 눈에 보이는 입출력이 아닌, 가장 핵심적인 연산 코어부터 바깥쪽으로 확장해 나가는 상향식(Bottom-up) 설계 방식을 채택함.

* **Core:** 시스템의 존재 이유 파악 👉 두 배열을 곱하고 더하는 MAC 연산 엔진 구축
* **Defense:** 치명적 변수 차단 👉 부동소수점 오차로 인한 판정 오류 방지 (Epsilon 비교)
* **Parser:** 데이터 정제 👉 라벨 정규화 및 키 파싱 로직 분리 설계
* **I/O & Report:** 입출력 및 가치 증명 👉 터미널 I/O, JSON 자동화, O(N²) 시간 복잡도 검증

## 2. 개발 로드맵 및 진행 상황
* ✅ Phase 1-1. 심장 만들기: 이중 for문을 이용한 MAC 연산(`calculate_mac`) 로직
* ✅ Phase 1-2. 오차 방어막: 부동소수점 오차 무시 및 동점(`UNDECIDED`) 판별 로직
* ✅ Phase 2. 데이터 전처리: 라벨 통일(정규화) 및 키 파싱 기능
* 🔲 Phase 3. 입출력 조립: 모드 1(사용자 입력) 및 모드 2(JSON 로드/예외 처리)
* 🔲 Phase 4. 성능 분석: `time.perf_counter()` 기반 연산 시간 측정

## 3. 기술 원리 및 심층 분석 (진행하며 채워넣을 예정)

* **부동소수점 오차와 Epsilon 비교 정책 도입 사유**
